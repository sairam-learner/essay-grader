# 🧠 AI-Powered Essay Grader

An intelligent web application that automatically evaluates essays using Natural Language Processing (NLP) and Deep Learning techniques. The system provides instant scoring, grammar feedback, and semantic analysis to improve writing quality.

---

## 🚀 Features

* ✍️ Automated essay scoring
* 🧾 Grammar and spelling correction
* 🧠 Semantic analysis using NLP
* ⚡ Real-time feedback with interactive UI
* 📊 Easy-to-use Streamlit interface

---

## 🛠️ Tech Stack

* **Language:** Python
* **Frameworks & Libraries:** TensorFlow, BERT, spaCy, Streamlit
* **NLP Tools:** NLTK / spaCy
* **Frontend:** Streamlit

---

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/sairam-learner/essay-grader.git
cd essay-grader
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```

---

## 💡 How It Works

* User inputs an essay
* Text is preprocessed (tokenization, cleaning)
* NLP model analyzes grammar and meaning
* Deep learning model predicts score
* Feedback is displayed instantly

---

## 📸 Output

* Essay score (based on content quality)
* Grammar suggestions
* Improvement feedback
